Tentang RPL

SMK Negeri 1 Dlanggu mengeluarkan produk unggulan dalam bidang pendidikan, yang berbasis IT yaitu Rekayasa Perangkat Lunak. Suatu bidang pendidikan yang mempelajari tentang Pemrograman, baik pemrograman aplikasi maupun pemrograman web.Rekayasa Perangkat Lunak SMK Negeri 1 Dlanggu berdiri pada tahun 2009.

Berikut materi-materi pemrograman yang ada di jurusan Rekayasa Perangkat Lunak :
RPL
Materi Visual Basic
SQL Server
C++
Delphi
HTML
